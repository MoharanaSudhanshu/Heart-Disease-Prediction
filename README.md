# ❤️ Heart Disease Prediction System

A Machine Learning based Heart Disease Prediction Web Application built using **Python, Flask, Scikit-learn, HTML, CSS, and JavaScript**.

The system predicts the likelihood of heart disease based on patient health parameters and provides instant results through an interactive web interface.

---

## 🚀 Features

- Heart disease risk prediction using Machine Learning
- User-friendly and responsive interface
- Flask backend integration
- Logistic Regression based prediction model
- Real-time prediction results
- Interactive web pages with modern UI

## 📊 Dataset

The model is trained using the Heart Disease dataset containing medical attributes such as:

- Age
- Gender
- Chest Pain Type
- Resting Blood Pressure
- Cholesterol
- Fasting Blood Sugar
- Rest ECG
- Maximum Heart Rate
- Exercise Induced Angina
- Old Peak
- Slope
- CA
- Thal

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/MoharanaSudhanshu/Heart-Disease-Prediction.git
cd Heart-Disease-Prediction
```

### Create Virtual Environment

```bash
python -m venv venv
```

Activate:

Windows

```bash
venv\Scripts\activate
```

Linux / Mac

```bash
source venv/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

---

## ▶️ Run Application

```bash
python app.py
```

Open:

```text
http://127.0.0.1:5000
```
